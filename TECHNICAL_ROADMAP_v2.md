# HandicapLab Technical Roadmap v2

This document charts the roadmap of the HandicapLab platform toward SaaS product release.

---

## Roadmap Milestones

| Version | Milestone | Scope / Deliverables | Status |
| --- | --- | --- | --- |
| **v1.0** | **Research Platform** | Poisson models, Kelly v2, Event-driven shadow paper trading, CLV observer. | ✅ Complete |
| **v1.1** | **Live Odds Platform** | Websockets, adapter interfaces, consensus, downtime checks, rate limiting. | 🔜 Sprint 26 |
| **v1.2** | **Multi-Bookmaker Consensus** | Dynamic normalizers, bookmaker agreement, market quality index scoring. | ⏳ Planned |
| **v1.3** | **Feature Store** | Versioned features, Parquet exports, retraining loop datasets. | ⏳ Planned |
| **v1.4** | **Portfolio Optimizer** | Bankroll management simulations, Kelly fractional adjustments. | ⏳ Planned |
| **v1.5** | **Public API** | Secured developer gateways, REST/WebSocket API feeds. | ⏳ Planned |
| **v2.0** | **SaaS Production Release** | Stripe/Midtrans monetization, OAuth, dashboards, notification alerts. | ⏳ Planned |
