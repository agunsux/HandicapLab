# SHADOW MODE DASHBOARD USER & OPERATOR GUIDE

**Route:** `/shadow-mode`

---

## Overview

The Shadow Mode Dashboard is designed with a high-density, Bloomberg Terminal aesthetic to provide quantitative researchers and operators with transparent real-time metrics.

### Key Navigation Views:
1. **01 // OVERVIEW & HERO METRICS**: High-level ROI matrix across 7, 30, 90, and 365 days, CLV tracking, ECE calibration error, and league breakdowns.
2. **02 // PREDICTION EXPLORER**: Filterable, inspectable feed of every prediction snapshot recorded before kickoff with cryptographic SHA-256 hash chains.
3. **03 // CALIBRATION & RELIABILITY**: Interactive reliability curve displaying expected vs observed probabilities across 10 probability buckets.
4. **04 // DRIFT & INTEGRITY**: Real-time PSI drift monitoring across feature, prediction, probability, market, and league dimensions.
5. **05 // WEEKLY REPORTS**: Exportable weekly scientific audit reports in markdown and JSON format.
